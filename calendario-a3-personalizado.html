Preguntas frecuentes

¿A3 vertical o apaisado: cuál elijo?

Vertical si quieres más altura para fotos y notas. Apaisado si prefieres una composición más panorámica.

¿Cómo lo pido en copistería?

Pide “impresión A3 a color”, escala 100% y el tipo de papel. Si quieres, añade encuadernación.

¿Qué papel recomiendas?

Para un buen resultado, 170–250 g. Si es para pared, puedes plastificar o laminar.

¿El PDF está listo para imprimir?

Sí. Exporta y revisa una página de prueba a escala 100%.

¿Puedo usarlo como regalo?

Sí. Añade portada y fotos para un calendario personal y listo para regalar.
